# 📖 Weather App: <a name="about-project"></a>

> Weather App is a web application that allows users to search for weather data by entering city name or using their location. It is built with HTML5, CSS3, Javascript, Bootstrap, Geolocation API, and OpenWeatherMap API.

<!-- Features -->

### Key Features <a name="key-features"></a>
- Responsive layout
- Get weather data from OpenWeatherMap API
- User-friendly interface
- Users can search for weather data by entering city name.
- Users can use their location to get weather data.
